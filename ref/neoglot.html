neoglot.py¶

The primary executable file for Neoglot. Uses click to provide a command-line interface.

The function run() is not picked up by Sphinx because of click, but it is the actual command which the CLI runs.

neoglot.gen_syll()[source]¶

Generates a syllable with a random syllable structure.

neoglot.gen_word(minsylls, maxsylls)[source]¶

Generates a word with the specified number of syllables.

neoglot.print_words(count, minsylls, maxsylls)[source]¶

Prints a number of valid words.
