parse.py¶

Contains functions related to interpreting the Neoglot DSL.

The interface essentially involves creating an instance of Parse, which will automatically interpret the given Neoglot file.

class parse.Parse(file)[source]¶

Wrapper for the parse module that automatically runs all the parsing functions in __init__() and stores the parsed data in its internal variables.

parse.parse_category(definition)[source]¶

Reads a phoneme category and stores it in CATEGORIES

parse.parse_definitions(lines)[source]¶

The basic parse loop. Steps through each line of the language file and identifies what sort of definition it is, then passes it off to the appropriate parsing function.

parse.parse_name(definition)[source]¶

For a given line, returns an array containing the name defined at that location, and the contents of the definition.

parse.parse_syllable(definition)[source]¶

For a given syllable definition, reads it and stores it in SYLLABLES.

Syllables are stored as nested lists, each interior list representing a valid phoneme or phoneme group for that section of the syllable. If a phoneme is optional, the list includes the empty string.

parse.pull_elements(syllable)[source]¶

For a given syllable definition, returns a list of all bracket- or parenthesis- bounded elements.

parse.vomit(error)[source]¶

Shortcut error-raising function to reduce boilerplate.
